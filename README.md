# react 后台管理系统 demo

## `react-backgrounds` 前端

- 主要技术栈：`react + redux + react-router + antd + axios + scss` ​

### 页面

- 首页
- 用户管理
  - 用户列表
- 表格
- Excel
  - 导出 Excel
  - 上传 Excel
  - Zip
- 文本编辑器
  - 富文本
  - MarkDown
- 作品集
- UI 组件
  - 弹出层
  - 按钮
  - Typography 排版
  - 选择器
- 可视化大屏
- 动态
  - 动画
  - 文字
- 日志
  - bug 收集

## `servers` 服务端

- 技术栈： `node + express + json-server`

### 功能

- 用户注册
- 用户登录
- 用户注销
- 用户信息管理
- 作品集

## bash

- `git clone https://github.com/lp-Imagine/react-node-admin.git`

- `cd react-backgrouds`

- `yarn/npm install`

- `npm start`

- `cd servers`

- `node server`

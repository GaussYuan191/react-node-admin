## 后台管理demo--服务端

### 技术栈
`nodeJs + express + json-sercer + jsonwebtoken`

### 功能

- 用户注册
- 用户登录
- 用户注销
- 用户信息管理
- 作品集

### bash

- `git clone https://github.com/lp-Imagine/react-node-admin.git`

- `cd servers`

- `node server`
